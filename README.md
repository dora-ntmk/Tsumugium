# Tsumugium

The Japanese TTS Discord Bot using VOICEVOX.

VOICEVOXを用いたDiscord読み上げBotです。

Discordからのメッセージ取得、VOICEVOXでの音声生成、VC上での音声再生、jsonを用いた辞書登録機能のみをサポートしています。

# This repository in the development stages. このリポジトリは開発中です。

まだ現実的に使用できる段階ではありません。ご注意ください。

## 注意

Python 3.4.1以降での動作を見込んでいます。
それ以前のバージョンでは音声生成が失敗する可能性があります。

Discord側の仕様変更により、Discord.py v2.7.0以上が必要です。
